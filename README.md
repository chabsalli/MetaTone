# 📝 MetaTone

MetaTone은 하루의 경험을 기록하고, 그 안에서 드러나는 **소프트 스킬과 감정의 흐름**을 분석해  
자기 성찰과 성장을 돕는 Streamlit 기반 웹 애플리케이션입니다.

---

## 🚀 Features

- **Daily Reflection**
  - 하나의 입력 박스에서  
    `행동 → 감정 → 결과` 흐름으로 기록하도록 가이드 제공
  - 날짜, 증거/자료 링크(선택), 카테고리 입력 가능

- **Soft Skill Highlighting**
  - 작성한 텍스트 중 소프트 스킬이 드러나는 문장만 강조 표시
  - 경험 속 행동과 태도를 중심으로 분석

- **Growth Feedback**
  - 발전시키면 좋은 역량 제안
  - 각 역량별:
    - 연습/루틴 제안 2가지
    - 다음 기록에 도움이 되는 질문 2가지
  - 항목별 메모 저장 가능

---

## 🧭 How to Use

1. 앱에 접속합니다.
2. **오늘의 기록**을 작성합니다.
3. 기록을 제출하면:
   - 소프트 스킬이 드러나는 문장이 강조되고
   - 성장 방향과 질문이 자동으로 제안됩니다.
4. 제안된 내용을 다음 기록에 활용합니다.

---

## 🛠 Tech Stack

- **Framework**: Streamlit  
- **Text Analysis**: scikit-learn (TF-IDF)  
- **Language**: Python  
- **Deployment**: Streamlit Cloud  

---

## 🎯 Purpose

MetaTone은 정답을 제시하는 도구가 아니라,  
**개인의 경험을 구조화하고 돌아볼 수 있도록 돕는 기록 보조 도구**입니다.
